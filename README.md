📌 Problem Statement

Retail chains struggle to personalize offers due to lack of insights into customer behavior.

🧠 Solution Approach

We used K-Means Clustering to segment customers based on:

Annual Income
Spending Score
⚙️ Steps Performed
Loaded dataset using Pandas
Cleaned and inspected data
Selected relevant features
Applied clustering algorithm
Visualized clusters
Used Elbow Method to choose optimal clusters
Interpreted customer segments
📊 Key Insights (VERY IMPORTANT)

You can say:

Cluster 0 → High spending customers → target with premium offers
Cluster 1 → Low spending customers → target with discounts
Cluster 2 → Medium customers → target with personalized recommendations

💡 Business Impact
Improves personalization
Increases customer satisfaction
Boosts sales through targeted marketing
